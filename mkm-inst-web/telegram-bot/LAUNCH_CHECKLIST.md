## 🔧 **배포 명령어**

```bash
# 1. 환경 설정
cp env.example .env
# .env 파일 편집하여 실제 값 입력

# 2. 의존성 설치
npm install

# 3. 테스트 실행
npm test

# 4. 개발 모드 실행
npm run dev

# 5. 프로덕션 빌드
npm run build

# 6. 프로덕션 실행
npm start

# 7. 전체 배포 프로세스
npm run deploy
```
